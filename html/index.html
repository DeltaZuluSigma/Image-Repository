Command Description
login [username] [password] Logs into the image repository as the given user where if the user doesn't exist, the user is created.
  • username - The name of the user to log in as.
  • password - The password of the user to log in as.
logout Logs out as the current user from the image repository.
display [perm] Displays image names with the indicated permission.
Valid Permissions:
  • public - All public images.
  • private - All user-specific private images.
  • * - All public images and user-specific private images.
view [image_name] Displays the named image provided the user has appropriate permission (The image is public or is owned by the user).
[image_name] requires an image's full name including extention.
add [image_name] [perm] Establishes a given named image's permission and prompts the user for the image file.
Valid Permissions:
  • public - Viewable by any user.
  • private - Viewable only by the user who owns it.
[image_name] requires only the image's preferred name.
del [image_name] Deletes the named image provided the image is owned by the user.
[image_name] requires an image's full name including extention.
Note: For [image_name] to register a name, do not use spaces in it.
